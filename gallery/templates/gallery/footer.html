About This Project

This project was created as the final project in Harvard's CS50W (online version). It was heavily inspired by gallery sites of the 00's and 10's, such as DeviantArt and Elfwood. I wanted to bring the more comment and interaction focused interface into a more modern design concept.

Features include the ability to post singular image posts, prose posts, and blog posts, the ability to comment on and favorite posts of any kind, and the ability to follow other users. Comments can also be left on profiles. Profiles include avatars that appear across the site, custom titles, and a space for a longer bio. All large text fields (bios, comments, text sections of posts) allow for the use of the use of markdown for formatting.

The backend is built using Django with Python and SQLite. The frontend is built with just HTML (and Django's template language), a little JavaScript and whole lot of SASS.

If this site were to ever open to real users, there would be an algorithm implemented that would be a little different than what's typically found on other sites. It would encourage users to take their time in posting, by increasing visibility to posts after an absence of posting for more than a day. It would also focus more on comments and favorites given rather than interactions received. Algorithms should favor factors that users can control, and not punish users for things that are beyond their control.
